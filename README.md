# Установка

- Установите node.js

- У вас есть два варианта получить преоект : 

1. Клонировать проект: 
    1. Установите git
    2. Напишите в консоль `git clone https://github.com/VandiKond/Pomka-lite.git`
2. Скачайте .zip проект отсюда
